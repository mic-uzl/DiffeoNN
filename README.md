# Code for Diffeomorphism-Equivariant Neural Networks (DiffeoNN)
This is the full code used for the 2D Lung Segmentation in the publication "Towards Diffeomorphism-Equivariant Neural Networks via Canonicalization" by Oettinger et al. (https://openreview.net/pdf?id=hyK30n4mra). The provided code is for a 2D set up, but can be adjusted for 3D. 

When using the code, please cite "Towards Diffeomorphism-Equivariant Neural Networks via Canonicalization" and the papers cited for code in the publication.
